## About This Challenge

This repository documents my progress through the **21-Day SQL Challenge** - a structured program designed to build strong SQL foundations through daily learning, hands-on practice, and real-world problem-solving.

**Challenge Period:** November 3 - November 27, 2025  
**Sponsor:** [@DPDzero](https://twitter.com/dpdzero)  
**Host:** [@IndianDataClub](https://twitter.com/indiandataclub)

## Challenge Roadmap

### **Week 1: SQL Fundamentals**

| Day | Date  | Topic                                   | Key Concepts                                  |
| --- | ----- | --------------------------------------- | --------------------------------------------- |
| 01  | 03/11 | [Introduction to SQL & SELECT](Day-01/) | Basic SELECT, column selection, viewing data  |
| 02  | 04/11 | [Filtering with WHERE](Day-02/)         | WHERE clause, comparison operators, filtering |
| 03  | 05/11 | [Sorting with ORDER BY](Day-03/)        | ASC/DESC, multiple column sorting             |
